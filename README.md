# Rivolia Landing Page

Landing page comercial para Rivolia, enfocada en conversión y experiencia visual en desktop y móvil.

## Vista general

Este proyecto implementa una página promocional de producto con:

- Hero principal con CTA y versión adaptada para móvil
- Secciones informativas de beneficios y propuesta de valor
- Catálogo de productos con carrusel en móvil
- Sección de pasos de preparación
- Comparativa visual y testimonios
- CTA final y botón flotante de WhatsApp

## Stack técnico

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- ESLint

## Requisitos

- Node.js 20+
- npm 10+ (o gestor equivalente)

## Instalación y ejecución local

```bash
npm install
npm run dev
```

Abrir en navegador:

- http://localhost:3000

## Scripts disponibles

```bash
npm run dev     # entorno de desarrollo
npm run build   # build de producción
npm run start   # servidor de producción
npm run lint    # revisión con ESLint
```
